That prior run is not a strict apples-to-apples metric comparison because it
started from `/home/erikg/impg/data/c4_low_seqwish_k_20260605T140000Z/graphs/c4.k311.poa2kb.gfa`,
not from the raw k311 seed. Still, the local cleanliness signal is clearly
worse for abPOA10k from the seed.

The prior global SPOA motif result had 9,189 segments, 12,124 links,
3,477,737 path steps, 260,568 total segment bp, 25 direct self-loop edges,
41,312 adjacent same-node path steps, and 4,421 self-loop repeat runs after
three rounds. The abPOA10k k311 seed output has comparable direct self-loop
and adjacent-repeat residual counts, but at 178,142 segments, 181,283 links,
80,156,435 path steps, and 280,038 total segment bp. In other words, abPOA10k
is not locally cleaner here; it preserves paths but over-fragments the graph
while leaving substantial loop motifs behind.
