## Policy note

The path-copy scaffold-glue guard added here is a provisional performance
boundary, not the final biological frequency-mask policy. It prevents high-copy
local syncmers from generating an all-pairs scaffold-context explosion while
preserving sequence as private unfolded occurrences.

The desired next step is spectrum-driven calibration: for each local syng graph,
record node occurrence count, path/haplotype count, occ/path ratio bins, max
occurrences per path, and dispersion where cheap, then compare candidate
thresholds on C4 and non-C4 controls. WG follow-up
`calibrate-spectrum-driven` tracks that calibration work.
